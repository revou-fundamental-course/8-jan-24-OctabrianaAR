Sebuah web apps sederhana yang berfungsi untuk menghitung nilai luas dan keliling persegi dan persegi panjang

Luas Persegi

Rumus Luas Persegi Yaitu :

L = S x S

Di mana :

L = Luas

S = Sisi



Hitung Luas Persegi

Keliling Persegi

Rumus Keliling Persegi Yaitu :

K = 4 x S

Di mana :

K = Keliling

S = Sisi



Hitung Keliling Persegi

Luas Persegi Panjang

Rumus Luas Persegi Panjang Yaitu :

LPP = P x L

Di mana :

LPP = Luas Persegi Panjang

P = Panjang

L = Lebar



Hitung Luas Persegi Panjang


Keliling Persegi Panjang

Rumus Keliling Persegi Panjang Yaitu :

K = (2 x P ) + (2 x L)

Di mana :

K = Keliling

P = Panjang

L = Lebar



Hitung Keliling Persegi Panjang
